    녹색 채움(width = bar%). 기본값 행은 배경 틴트 + 녹색 라벨 + "◀ 시작값" 표시 + 밝은 막대.
  - 막대 의미가 형질마다 다른 곳(속도=최고속 대비, 공격력=성공률, 무리=감소율)은 note 한 줄로 설명.

## 결정 사항
- **막대는 "실제 인게임 크기"를 0~1로 정규화.** 예: 속도 0.5 → 막대 65%(절반 아님). 0.5가
  값의 절반이 아니라는 비선형을 막대가 그대로 보여줘 "형질 0.5 = 효과 절반"이라는 오해를 깬다.
  값 텍스트(약 46 등)와 note 로 기준을 명시해 막대 의미 차이를 보완.
- **기본값 행에 "◀ 시작값" 명시.** 모든 종이 0.5에서 시작한다는 핵심을 표 안에서 바로 읽게 함
  (카드 +0.22 가 0.5→0.72 라는 걸 도감과 연결).
- 막대는 형질 도감(수치형)에만. 생물 도감(식성·시작 수 등 범주형)·위협 도감(약점 텍스트)엔
  의미가 없어 추가 안 함.
- 동시 세션 없음(ACTIVE_WORK 레인 전부 비었음/보류). glossary.ts 단독 작업이라 무겹침.
