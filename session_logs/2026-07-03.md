## 결정/기록
- **날개 = swimming의 완전한 대칭.** 산·물 통행(canFly) + 고산 먹이 틈새(mountainous, canFly만) + 카드
  (지느러미↔날개) + 프리셋(바다 개척자↔하늘 개척자). "형질이 조건부로 빛남"(산·바다 많은 맵=비행 유리).
- **야생 고산 종은 이번에 제외.** `WILD_ARCHETYPES`에 종을 더하면 `generateWildSpecies`가 `this.rng`를
  더 소비 → 그 뒤 spawnFood/spawnEntities rng가 밀려 전체 밸런스 이동. 고산 먹이는 플레이어 비행 종의
  완전 독점 틈새로 두는 게 더 깔끔. 야생 고산 종은 나중에 kin식 독립 rng 스폰으로(backlog).
- **비행 대사 대가는 프로브로 0.45→0.25.** 0.45는 과해 여러 맵(env-2·m1·sky-1)서 비행 종이 굶어 멸종.
  0.25에서 모든 맵 자생 + 맵별 유불리(육지 풍부=지상 유리, 척박·산바다 많음=비행 유리)로 균형. 압도·멸종 없음.
- **밸런스 함정 발견·기록(known_issues).** cohesion의 lineOfSight 호출에 canLand를 명시하니 물 전용 종
  (swimming≥90) 동작이 바뀌어 독안개 통과기준이 경계 이동. 고산 먹이를 0으로 꺼도 실패해 원인이 그게
  아님을 가렸다. → 원래 호출(canSwim만) 보존. "새 파라미터 배선 시 기존 기본값 의존 호출은 최소 변경".

## 세션 3 — P5 전투 형질 (독침 + 원거리)

### 요약
ROADMAP P5 착수 — 전투 형질 2종(GENOME v6). 원거리(ranged)는 사거리 확장으로 명확히 작동(선제 사냥,
ranged100→사거리 34로 자생 크게↑), 독침(venom)은 "물면 독 걸려 약해지고(보라 tint·DoT) 잡기 쉬워진다".
echo/wings 미러링(맨 끝·기본 0·rng 없이)이라 기존 밸런스 보존. 94 테스트 통과.

### 주요 변경
- `genome.ts` v6: venom·ranged(TRAIT_KEYS 맨 끝, 기본 0). migrate v1~v5→v6.
- `entity.ts`: poison(누적 독)·poisonSource(독 건 포식자 — 독사 시 에너지 획득). `world.ts` DeathCause venom.
- `behavior.ts`: 사거리 = attackRange + ranged01×rangedBonus. venom 물면 prey.poison 누적 + 즉사 확률에
  poison 기여(약해진 먹이 잡기 쉬움) + 매 틱 DoT. `params.ts`: rangedBonus 22·venomOnHit 8·venomTickDamage
  1.5·venomKillBonus 2.5(프로브 튠).
- `species.ts`: venom·ranged rng 없이 + 친척 전투 정체성. `cards.ts`: 독니·독샘·긴 뿔·독침 뱉기 카드 +
  독 사냥꾼 프리셋(7종). `worldView.ts`: 독니(보라)·긴 뿔 외형 + 독 걸린 개체 보라 tint.
- UI: runReport DEATH_LABELS 중독 + 형용사, glossary 독침·원거리 도감, traitPanel EDITABLE. hud CAUSE_LABEL.

### 결정/기록
- **독침 재설계(known_issues)**: 순수 DoT 는 소수 개체+즉사 판정 사냥과 충돌(즉사가 독보다 먼저 죽여
  중독사 0, 즉사 끄면 무한 추격으로 멸종). → 독을 "즉사 확률 누적 기여"로. venom=누적 / attack=즉발
  트레이드오프. DoT 는 약하게(도망친 먹이 마무리 + 보라 tint 시각). 순수 중독사는 포기.
- **전투 형질은 자생 개체수에 미미**: base killChance 0.5 가 커서 사냥이 이미 잘 되고, 개체수는 채집+
  번식이 주결정. venom 유무 자생 차이 작음(원거리는 사거리로 명확). 밸런스 세부는 폰 체감 후.
- **poisonSource 로 독 종 자생 보장**: 독으로 죽은 먹이의 에너지를 마지막에 문 포식자가 얻는다(즉사와
  동등) — 없으면 독 종이 사냥해도 굶는다.
