# 2026-06-27 세션 로그

## 세션 — preview 를 실제 플레이와 같은 비율로
사용자: "고정 px UI라 그런지 preview 에서 보는 거랑 실제 웹 플레이 환경이 다르게 생겼다."

### 원인
- HUD·HTML 패널은 화면 픽셀(CSS px) 고정 → UI 가 차지하는 비율은 **뷰포트 폭**에 달림.
- preview 의 데스크톱 iframe 이 960 이라 실제 데스크톱(1440~1920)보다 작음 → 같은 고정 px UI 가
  preview 에선 화면의 ~25%, 실제론 ~12~16% → preview 에서 과대하게 보였다.

### 주요 변경 (`public/preview.html` 만)
- iframe 내부를 **실제 기기 CSS 크기**로: 폰 390×844, 데스크톱 **1440×810**(실제 데스크톱 비율).
- 화면엔 `transform: scale()`(폰 0.9, 데스크톱 0.52)로 **축소만** 표시(`.screen` 래퍼 + overflow:hidden,
  transform-origin 0 0). 앱은 실기 크기로 렌더 → preview 비율이 실제 플레이와 일치.
- 라벨에 실제 CSS 크기·축소율 표기.

### 결정/기록
- **고정 px UI + transform:scale 이 정답.** 앱의 innerWidth 는 iframe 내부 크기(=실기 크기)를 보고,
  화면 표시만 축소 → 비율 보존하며 나란히 배치. 클릭/탭은 브라우저가 변환 좌표 처리(정상).
- 데스크톱은 1440 고정(예측 가능). 실제 1920 이면 UI 가 더 작아지지만 960 대비 훨씬 근접.
- 동시 세션(glossary 등)과 무겹침: preview.html 단일. src/ 무관.

### 미해결 / 다음
- (이월) 데스크톱 추가 최적화, 형질창 크기 미세 조정.

## 세션 — 메커니즘 가독성(렌더) + 대백과 (remote-control)

### 요약
"형질·메커니즘이 게임 안에서 이해가 안 된다"는 피드백을 두 갈래로 풀었다. (1) 보이는 것을 실제
메커니즘에 맞추는 렌더 개선(행동/게이트는 안 건드림), (2) 용어 사전을 대백과로 키워 형질·생물·
위협·튜토리얼을 실제 수치와 함께 보게 했다.

### 주요 변경
- `render/worldView.ts` (314b73f) — 시야 반경 표시(내 종 일부에 파란 원), 독안개=전체 화면 녹색
  안개(전역 효과를 전역 그래픽으로), 무리/약탈자/외톨이 보스=작은 점 11개가 일렁이는 떼.
- `ui/glossary.ts` 신설→재설계→확장 (2c6a71b, b23fa82, ee9e31c) — "용어 사전"을 "대백과"로.
  2단계(목록→상세, SVG 그림 + 설명). 형질 도감은 0~1 범위·기본 0.5·카드 폭(0.15~0.28) 설명 +
  형질별 실제 게임 수치 표(속도 20/46/71, 시야 52/117/182, 사냥 50/76/24%, 보온 27/55% 등).
  생물 도감(내 종+야생 6종), 위협 도감(보스5·대멸종4 약점), 처음이라면(튜토리얼).
- `ui/lobby.ts`·`ui/controls.ts` — "대백과" 버튼(로비·일시정지)에서 열기 + main 배선.

### 결정/기록
- **전역 위협은 행동(도망)으로 풀지 않는다.** 독안개를 피해 다니게 하면 빠른 종이 빠져나가 "대사
  게이트"가 "속도 게이트"로 바뀐다. 대신 "독이 전역임"을 전체 화면 안개로 보여줘 죽는 위치가 자연
  스럽게 읽히게 했다. (독안개는 처음부터 globalDrain=전역이었고, 동그란 aura 그래픽이 오해를 산 것.)
- **카드 수치가 와닿으려면 기준이 필요하다.** "속도 +0.22"는 기본 0.5를 0.72로 올린다는 뜻 +
  그 값이 실제 게임에서 얼마인지(표)를 함께 보여야 의미가 생긴다 → 형질 도감의 핵심.
- **em dash 절대 금지(재확인).** 사용자 재지적. UI 문구 + 코드 주석까지 0건 검사로 확인.
- 동시 세션이 hud.ts·panelStyles.ts·preview.html 을 많이 건드리는 중 → glossary/controls/lobby/main
  만, 내 파일만 경로 스테이징, 타입체크는 타 세션 파일 필터링해 확인.
