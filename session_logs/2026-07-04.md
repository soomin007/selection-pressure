# 2026-07-04 세션 로그

## 요약
ROADMAP "야생 진화 심화"의 다음 단계 착수·완료 — 야생종이 자기 무리가 겪는 **압력별로** 적응한다.
기존엔 추위→고대사 하나뿐이었는데, 먹이 부족→저대사(효율), 포식자 노출→속도·무리(빠르고 뭉쳐 산다)를
더했다. 대사는 추위(위)와 먹이 부족(아래)이 밀당한다. 그리고 진화가 **폰에서 보이게** — 야생 텍스처
키에 거친 게놈 서명을 붙여, 형질이 크게 움직이면 스프라이트가 따라 바뀐다(전엔 종 id 고정이라 진화가
겉모습에 안 나타남). 99 테스트 통과(+4), 통과기준 밸런스 보존.

  먹이부족→저대사, 추위와 먹이부족의 대사 밀당.

## 결정/기록
- **적응 로직을 순수 함수로 분리한 이유**: 방향성(포식→속도↑ 등)을 결정론적으로 정확히 검증하려면
  라이브 시뮬(사냥·이동 간섭, 노이즈)이 아니라 압력 입력을 직접 준 단위 테스트가 필요. 측정(world)·
  적응(순수)·드리프트(rng)를 3분리해 각각 견고. known_issues의 "메커니즘 작동으로 테스트" 원칙 실천.
- **밸런스 보존 설계**: (1) 굶주림 끌기는 평균 에너지 < 0.35 에서만 켜져 배부른 시련 상황은 기존 대사식
  그대로. (2) 포식 적응은 목표=현재+노출비율×범위를 wildAdaptRate(0.05)로 천천히 따라가 짧은 시련
  (750~2000틱=2~6주기)엔 몇 점만, 긴 런(6000틱=20주기)엔 10~30점 이동. 99 테스트 통과가 이를 확인.
- **프로브로 강도 확정**(4시드×6000틱): 피식자(작은 풀벌레 spd 37→55·hd 75→88, 물고기 떼 46→62·53→76)는
  빨라지고 뭉치고, 최상위 포식자는 노출 적어 덜 오르고, 포식 없는 무리(느린 거북 20→19)는 안 오른다
  (다양성 유지). 추운 s3 는 대사 전반↑(40→66), 따뜻한 env-1 은 유지/하락. pop 37~65 건강(폭발·멸종 없음).
- **진화를 보이게 = 텍스처 갱신이 필수**: 야생은 공유 게놈이 진화하는데 텍스처가 종 id 로 고정이라 여태
  겉모습이 안 변했다(진화가 화면에 없음 = visible-feedback 위배). 거친 12-버킷 서명으로 "눈에 띄는 변화만"
  새 스프라이트 → 캐시 폭증 없이 진화가 폰에 드러난다.
