# 적자생존 — 화면 소개 스크린샷

한 종(種)을 길러 생태계 정점에 올리는 **관전형 진화 로그라이크**.
플레이어는 직접 조작하지 않고, 라운드 사이에 카드로 형질을 **고르기만** 한다.
규칙 기반 생태 시뮬이 그 결과를 살아 움직이게 한다. 모바일·세로 우선(TypeScript + PixiJS).

아래 9장은 `node scripts/screenshots.mjs` 로 자동 캡처한 것(고정 시드 `showcase-1`).
01~07·09 는 모바일 세로(440×940), 08 은 데스크톱 가로(1280×720).

## 톤·아트 메모 (디자인 참고)
- **아트 방향:** 둥근 몸통 + 큰 눈의 단순한 생물, 등에 톱니·주둥이 같은 형질이 실루엣으로 드러난다.
  타일 기반 바이옴(풀·물·모래·바위)이 배경. 색은 어두운 바탕에 형질별 포인트 색.
- **문체:** 사용자에게 보이는 한국어는 "차분한 자연 관찰 다큐멘터리" 톤(현재형 관찰, 담백, 쉬운 말).
  한글 사이 em dash 금지, 전문 용어는 풀어 쓴다.
- **재생성:** 화면이 바뀌면 `node scripts/screenshots.mjs` 로 다시 뽑는다(같은 시드라 거의 동일하게 재현).
