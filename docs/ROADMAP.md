# 로드맵 (앞으로 할 일만)

> 기획서 §9 기준. 각 단계 사이에 명시적 리뷰 체크포인트(폰 URL 검토). 완료 항목은 지운다.
> Phase 5 전까지 코어 싱글플레이 루프가 다른 플레이어 0명으로도 재밌어야 한다.

| Phase | 내용 | 검증 기준 |
|---|---|---|
| **S1** | (스트레치) 메타 언락 | 수평 확장, "한 판이 재밌다" 확인 후 |
| **S2** | (스트레치) 비동기 생물 | 수동 큐레이션 → 추후 서버 자동화 |

> Phase 0~6 + 다종 생태계까지 코어 완성. 이후는 폴리시/스트레치 + backlog 의 이월 항목.

## 완료
- **Phase 0** — 기획 확정 + 스캐폴드(TS/Pixi/Vite, 시드 RNG, 버전 게놈, scale-to-fit).
- **Phase 1** — 시뮬 코어(월드/개체/먹이/규칙기반 행동: 채집·허기·죽음·번식, 고정 타임스텝
  결정론, 멸종 시 재시작, 개체 수 HUD).
- **Phase 2** — 게놈/형질(환경 시드와 게놈 분리, 형질 슬라이더 패널: 즉시 적용 + 같은 환경
  재시작 비교, 형질에 따라 개체 수가 뚜렷이 갈림을 확인).
- **Phase 3** — 절차 환경(추위/비옥도 필드, 맵마다 다름, 배경 시각화 + 개체 수 추이 그래프).
  추운 맵=고대사 유리 / 따뜻한 맵=저대사 유리로 환경이 유리 형질을 뒤집음을 확인.
- **Phase 4** — 카드 드래프트 메타루프(드래프트→관전 반복, 카드=형질 누적, 멸종 시 런 종료,
  결과 화면+새 런, 환경 요약 표시).
- **Phase 5** — 보스 게이트 + 대멸종 피날레(위협 예고, 보스 3종=속도/번식·건강/저대사,
  대멸종 3종=한파/폭염/대가뭄 적합도 필터, 승리·패배 판정). 게임성 1차 완성.
- **다종 생태계** — 내 종 + 야생종(초식 경쟁자·들풀 무리·포식자), 식성(육식/초식) + 공격성(사냥/전투)
  + 포식자 회피(속도) + 먹이 경쟁 연결. 공간 격자로 이웃 질의 O(n). 승패는 내 종 기준.
- **식성 선택 + 잡식 + 형질 스프라이트 + 데스크톱 UI** — 시작 식성 선택(초식/잡식/육식),
  잡식 지속 가능. 생물=게놈 기반 스프라이트(몸/눈/가시/주둥이)로 크게. 데스크톱은 가로+카드 한 줄.
- **게임 셸** — 로비/타이틀, 멈춤 메뉴, 배속(1/2/3x), 관전 시간 단축, HUD 선명화(스케일 밖).
- **Phase 6 가독성** — 사망 원인 집계+정산 막대+라이브 피드, 하이라이트 배너(보스/대멸종/멸종 위기/
  관문 통과), 보스 추적 줌 카메라, 보스 주목 펄스.
