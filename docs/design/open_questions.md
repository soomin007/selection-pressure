# 미해결 결정 (기획서 §11)

> 구현 진행에 막히는 시점에 하나씩 확정한다. 확정되면 기획서 본문/백로그에 반영하고 여기서 지운다.
> 일부는 기본값을 제안해 두었으나 확인 필요.

1. **화면 방향(orientation)** — *제안: 세로(portrait) 기본* (현재 스캐폴드도 세로 540×960).
   넓은 사바나 부감은 가로가 유리. 레이아웃 전체를 좌우하는 결정.
2. **승리 조건 수치** — 보스 개수, 런 길이(분), 라운드 수. (참고: Everything is Crab ≈ 20분 / 보스 3마리)
3. **플레이어 최적화 목표 정밀 정의** — "정점 포식자"를 무엇으로 측정할지 (천적 멸종? 정점 유지 N틱?
   보스 통과만으로 충분?).
4. **플레이어 생물 프레이밍** — 정점 라이벌 게이트 vs 런 중 야생 경쟁종 (기획서 §6.3 중 택1).
5. **형질/카드 구체 목록** — Phase 2 진입 전 별도 패스로 작성.
6. **타이틀 확정** — 현재 코드명 "적자생존". 영문/배포명은 "Selection Pressure"(폴더명) 사용 중.
