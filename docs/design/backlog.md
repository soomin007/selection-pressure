## 미니맵 · 바다 맵 (맵 확장 3배 완료 → 후속)
- [ ] **미니맵** — 큰 맵(3배)에서 내 무리·지형(바다/육지/산)·위협 위치를 한눈에. 카메라가 무리를
  따라가 화면엔 일부만 보이므로 전체 조망용으로 거의 필수. 좌상단/우상단 작은 패널.
- [ ] **물 전용 플레이어 본격화(바다 맵)** — 바다 개척자를 물전용(swimming≥0.9)으로 되돌리려면 바다가
  넓어야 한다(현재 맵 12%라 이주 없는 플레이어는 구조적 전멸 → 지금은 양용 0.85 로 둠). 맵 확장 때
  바다 비율↑(바다 맵) + 바다 먹이 풍부 + 바다 생물 전용 특성을 함께. 물전용 보정 코드
  (`aquaticInitialEntities`·보금자리 바다화)는 `world.ts` 에 이미 있음(양용이라 비활성 대기).
