## Phase 3 — 절차 환경 (다음 착수 대상)
검증 기준: 판마다 다른 맵이 생성되고, **환경마다 유리한 게놈이 달라진다**.
(토대 있음: 환경 시드가 게놈과 분리됨. 식량 희소도가 시야/속도 가치를 바꾸는 것도 이미 확인.)

- [ ] 절차 맵 생성: 구역(바이옴)별로 식량 분포·위험·기후가 다르게
- [ ] 환경→유리 형질 연결: 예) 추운 구역=대사 유리, 개활지=속도/시야 유리
- [ ] 환경 시각화: 지형/식량대/위험 구역이 화면에서 구분되게
- [ ] 같은 게놈을 다른 환경에 넣으면 결과가 갈리는지 테스트 추가
