## Phase 2 — 게놈/형질 (다음 착수 대상)
검증 기준: 게놈을 바꾸면 생존 결과가 **눈에 띄게** 달라지는 것이 화면에서 보인다.
(이미 시드마다 종 게놈이 달라 전개가 갈리는 건 확인됨. Phase 2 는 "의도적으로 바꿔 비교"로.)

- [ ] 종 게놈을 런타임에 조정하는 디버그 패널(슬라이더) 또는 프리셋 비교 모드
- [ ] 형질 → 결과 가시화: 시야 반경/속도 등을 화면에 표시(선택적 디버그 오버레이)
- [ ] `diet`(식성) 형질 행동 연결: 육식 성향이면 다른 개체/먹이 우선순위가 달라지게(초보적)
- [ ] 형질별 균형 점검: 한 형질만 극단으로 줘도 생존 가능한지(단일 정답 형질이 없게)
