## 다음 세션 바로 후보 (2026-07-06 갱신 — 최근 대량 변경 후 상태)
- [ ] **보스 아트 통합 (사용자 이미지 대기)** — 사용자가 `docs/design/boss_art_prompts.md` 프롬프트로 AI 이미지
  생성 중. 받으면 `render/worldView.ts`의 절차 도형(drawBossCreature)을 스프라이트 로드로 교체(타입별 텍스처
  캐시, 진행방향 회전, 물기 반경·맥동 유지). 통합 메모는 프롬프트 문서 하단 참고.
- [ ] **최근 변경 폰 검증·튜닝** — 한 번에 많이 바꿔 체감 확인 필요: 형질 200 스케일 성장 속도
  (`cards.ts` CARD_GROWTH_SCALE 0.6), 카드 수치=실제값 동기화, 프리셋별 카드 풀(cardRedundant), 언락 레벨
  임계(`meta.ts` 3/5/7/9/12), 시대 난이도 곡선(`config.ts` eraDifficultyStep 0.22), 챔피언 등장, 스킵 보상
  (draftSkipBrood 2), 승리·정복·멸종 연출 강도(momentOverlay), 수동 카메라·줌·"한 마리 관찰". 약하면/과하면 상수 조정.
- [ ] **큰 시스템 후보 (하나 골라 착수)**:
  - **개체별 진화(자연선택)** — ROADMAP 핵심. 종 공유 게놈 → 개체 게놈(새끼가 부모 닮되 변이, 못 버티면 솎임).
    "내 애들이 각자 다르고 진화한다"의 완성. sim 코어 대공사 + 밸런스 재조정(신중히, 프로브 필수).
  - **시대별 보상** — 시대를 넘을 때마다 보상(좋은 카드·형질). 난이도 루프를 "갈수록 재밌게"(현재는 어렵기만).
  - **EiC식 스킵 언락** — 드래프트 스킵을 여러 런 조건 달성 시 열리는 공통 형질로(사용자 언급).
- [ ] **날개(비행) 폰 튠** — 비행 대사 대가(flyMetabolismCost 0.25)·시야 보너스(flyVisionBonus 0.5)·
  고산 먹이(mountainFoodPatches 30) 체감 후 조정. 날개 스프라이트가 보이는지, 산 위 흰빛 고산 먹이가
  보이는지, 비행 vs 지상 트레이드오프가 재미있는지. 하늘 개척자 프리셋/날개·튼튼한 날개 카드로 테스트.
- [ ] **야생 고산 종** — 산 위에 사는 비행 야생종(바다 풀뜯이·물고기 떼가 바다에 생명을 주듯 산에도).
  단 `WILD_ARCHETYPES` 추가는 `generateWildSpecies`의 rng 스트림을 늘려 밸런스가 통째로 이동 → kin 처럼
  **독립 rng 스폰**으로 넣어야 한다(별도 설계). 넣으면 밸런스 프로브로 기존 분포 보존 확인.
- [ ] **전투 형질(방어독·원거리) 폰 튠** — 방어독 강도(venomOnHit 100·tickDamage 15 — 잡아먹으면 손해)·
  원거리 사거리(rangedBonus 22) 체감 후. 독 살갗으로 포식이 막히는지, 원거리 사냥꾼이 멀리서 치는지,
  독 걸린 포식자 보라 tint. 방어독이 너무 세면(포식 완전 회피) 강도↓.
- [ ] **산 보물 폰 튠** — mountainFoodPatches 10·treasureSpawn 3·재생 4배 체감 후. 보물 먹으면 동족이
  확 불어나는 대박이 재밌는지. 레벨업으로 바꿀지(현재 동족 증식). 야생 고산 종과 보물 경쟁도 여지.
- [ ] **야생 진화 심화 폰 튠** — 압력별 적응(포식→속도·무리, 먹이부족→저대사, 추위→고대사) 강도 체감 후.
  한 판(≈6000틱) 지켜보면 야생종 형질·스프라이트가 눈에 띄게 갈리는지(작은 풀벌레가 빨라지고 뭉치는지,
  추운 맵서 대사 오르는지). 약하면 wildAdaptRate/Range↑, 너무 세 밸런스 흔들리면↓. 프로브로 분포 먼저.
- [ ] **물고기 학교 + 독 중독 표식 폰 튠** — ① 시작 물고기 15가 학교로 뭉쳐 보이는지·붐버스트가 자연스러운지
  (심하게 사라지면 deepFoodPatches↑). ② 깊은 바다(진한 남청) 먹이가 얕은 청록과 구분되는지. ③ 독 살갗
  프리셋으로 포식자에 보라 오라·방울(중독)이 뜨는지·잘 보이는지. 육지 떼종(작은 풀벌레·들풀) 실체화도 여지.
- [ ] **초음파 폰 튠** — echoBase 130·카드 강도(echo+42 vision-24) 체감 후. 초음파 원(보라)이 잘 보이는지,
  눈 vs 귀 트레이드오프가 재미있는지. 초음파 야생종(두더지·박쥐형) 추가도 여지.
- [ ] **레벨업 속도(xpBase) 튠** — 먹이 경험치 → 레벨업이 폰에서 너무 빠른지/느린지 체감 후 조정
  (`game/config.ts` xpBase 24·xpPerLevel 14). headless 로 750틱 내 종 먹이 섭취율 프로브 가능하나 폰이 정확.

## 시련 실재화 (사용자 방향: 시련이 맵·개체·형질과 겉돈다 → 실재화)
> 2026-07-02 개체형 떼 5종 완료(swarm·raider·isolation·stalker는 실제 떼 개체, chaser 는 단일).
> 독안개만 진짜 전역 재난이라 화면 틴트 유지. 남은 후보는 아래.
- [ ] **외톨이 사냥꾼 카운터 밸런스** — 무리성 카운터가 개체 시뮬에서 약함(무리성의 성장 부작용과
  상충, 단일 시드 노이즈). 폰 체감 후 저항/위협 조정. 필요하면 "이웃 수 기반"을 다시 시도(정교화).
- [ ] **험지 감속 강도 튠** — 소수 개체+국소 보금자리라 감속이 세면 내 종 붕괴 → 온건하게(최대 15%)
  둠. 폰 체감상 효과가 약하면(속도 형질 이점 미미) 조금 올리거나, 험지가 내 종 보금자리를 덜 덮게
  스폰 회피. 수풀 감쇠 강도도 폰 체감 후 조정.
- [ ] **지형×형질 더 넓히기** — 수풀(시야)·험지(속도) 완료(0628623·0177f45). 늪(감속+먹이)·개활지
  (시야 보너스) 등 추가 여지. 산은 현재 완전 차단 → "느리게 통과"로 바꾸면 길찾기 대공사(신중히).
- [ ] **개체형 떼 시련 용어** — 이제 5종이 killRadius>0 이라 다 "보스"로 분류("보스 · 약탈자 무리"
  어색). 단일 강력=보스 / 떼=? 로 용어 재정리 검토(폰 피드백 후).
