## Phase 5 — 보스 게이트 + 대멸종 피날레 (다음 착수 대상)
검증 기준: 위협 예고 + 보스별 위협 축이 다름 + 마지막은 대멸종 적합도 필터.

- [ ] 보스 위협 축: 속도형/장갑형/무리전술형/독 중 여러 종류, 막마다 다른 축
- [ ] 전투 전 위협 예고(게놈/보스 → 쉬운 말 요약). 안 그러면 카운터 불가 RNG 벽
- [ ] 보스 게이트 통과/실패 판정(내 종이 보스를 이기거나 버티는지)
- [ ] 최종 = 대멸종 이벤트(환경 적합도 필터): 잘 적응한 개체군만 통과
- [ ] 승리(모든 게이트 통과)/패배(멸종) 연결, 결과 화면 반영
