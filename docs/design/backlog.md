## Phase 1 — 시뮬 코어 (다음 착수 대상)
검증 기준: 한 종이 정적 맵에서 채집/허기/죽음까지 그럴듯하게 살아가는 것이 화면에서 보인다.

- [ ] `sim/world.ts` — 시뮬 월드 상태(틱 카운터, 개체 목록, 먹이 분포)와 `step(dt)` 루프
- [ ] `sim/entity.ts` — 개체 구조: 위치·속도·에너지·나이 + 게놈 참조
- [ ] `sim/food.ts` — 정적 맵의 먹이 분포(시드로 배치)
- [ ] `sim/behavior.ts` — 규칙기반 행동: 채집(먹이로 이동)·허기(에너지 소모)·죽음(에너지 0/노화)
  - 무리짓기(boids/steering)·도망·전투는 Phase 1 후반 또는 Phase 2 로
- [ ] `render/worldView.ts` — 월드 상태를 Pixi 로 그림 (개체=점, 먹이=점), `main.ts` 플레이스홀더 교체
- [ ] sim 단위 테스트: 같은 시드 → 같은 N틱 후 월드 상태 (결정론 회귀 테스트)
