## Phase 4 — 카드 드래프트 메타루프 (다음 착수 대상)
검증 기준: 런 구조가 돈다. `[드래프트] → [관전] → [정산] → [드래프트] …`

- [ ] 라운드 구조/상태기계: 드래프트 → 시뮬 관전(시간/조건 종료) → 정산 → 반복
- [ ] 카드 데이터 구조: 형질 증감 등 효과(런 내 영구, 런 종료 시 리셋)
- [ ] 드래프트 UI: 3장 중 1장 탭 선택(랜덤 후보 = 운 요소). 형질 슬라이더 패널은 디버그로 격하/토글
- [ ] 라운드 종료 조건(시간 N초 또는 목표) + 정산 화면(이번 라운드 결과 요약)
- [ ] 카드 효과를 게놈에 누적 적용 후 `clampGenome`
