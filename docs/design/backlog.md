## Phase 6 — 가독성 / 연출 (다음 착수 대상)
검증 기준: "왜 이겼나/졌나"가 화면에서 더 잘 읽힌다.

- [ ] 死因 집계: 굶음/추위/폭염/보스/노화 별 죽음 수 → 라운드 정산에 표시
- [ ] 하이라이트: 주목할 사건(보스 등장, 멸종 직전, 급증) 자동 포착/표시
- [ ] 카메라 추적 또는 줌(군집/보스에 주목)
- [ ] 결과 화면에 최종 형질 요약("이 종은 어떤 종이었나") 표시

## 보스/형질 깊이 (이월)
- [ ] 시야를 카운터로 쓰는 보스 재설계(현재 titan 은 느려서 비활성)
- [ ] `diet`(식성)·`attack` 행동 연결(육식/전투) + 관련 카드 추가
      (`herding` 은 연결됨: 무리 이동 + 모이면 보온)
- [ ] 대멸종과 보스 위협이 충돌할 때(예: 독=저대사 vs 한파=고대사) 난이도 점검

## Phase 2~3 이월 (형질·환경 깊이)
- [ ] `diet`(식성) 행동 연결: 육식 성향이면 다른 개체를 사냥(초보적 포식)
- [ ] `herding`(무리 성향) → boids/steering(정렬·응집·분리)
- [ ] 환경에 위험(hazard) 구역 추가(현재는 추위/비옥도만)
